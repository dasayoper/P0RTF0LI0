## Название проекта
Restaurant API project

## Описание проекта
Учебный проект: API для сети пиццерий

## Функционал проекта:
* Регистрация/авторизация
* Бронирование столика
* Просмотр меню
* Оформление заказа
* Модуль администрирования

## Используемый стек технологий
* Spring Boot
* Security + JWT
* JPA
* Lombok
* Open API + Swagger
* Jupyter
* *...дополняется...*

## Базы данных
* PostgreSQL
* Redis
