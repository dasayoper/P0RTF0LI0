## Название проекта
Restaurant API project

## Описание проекта
Учебный проект: API для сети пиццерий (2021)

## Функционал проекта:
* Регистрация/авторизация
* Бронирование столика
* Просмотр меню
* Оформление заказа
* Модуль администрирования

## Используемый стек технологий
* Spring Boot
* Security + JWT
* Spring Data JPA
* Lombok
* Swagger(Open API)
* JUnit
* *...дополняется...*

## Базы данных
* PostgreSQL
* Redis
