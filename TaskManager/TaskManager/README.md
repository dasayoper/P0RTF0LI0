## Название проекта
Task Manager project

## Описание проекта
Учебный проект: API для планировщика задач (2024)

## Функционал проекта:
* Создание/просмотр/редактирование/удаление задач
* Создание/просмотр/редактирование/удаление комментариев к задачам
* Просмотр/изменение информации об аккаунте
* Регистрация/авторизация
* Для поиска реализованы фильтры и сортировка
* В системе есть разделение по ролям: администратор, пользователь

## Используемый стек технологий
* Spring Boot
* Security + JWT
* Spring Data JPA
* Lombok
* Swagger
* JUnit + Mockito
* Docker (check START.md for deploy info)
* Flyway for DB migrations
* Logback for file logging

## Базы данных
* PostgreSQL
* Redis
